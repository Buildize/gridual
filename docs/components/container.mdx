Container
=========

Container component makes the page or the component centered and padded
horizontally.

### Usage

```
import { Container } from 'gridual';

<Container>
  amazing content here
</Container>
```

### Props

| Name     | Type    | Description                               | Default |
|----------|---------|-------------------------------------------|---------|
| maxWidth | string  | sets the maximum width for the page       | 1180px  |
| fluid    | boolean | makes the maximum width 100%              | false   |
